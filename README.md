CHIP-8 Interpreter in C with SDL2

A simple and portable CHIP-8 emulator/interpreter written in C using SDL2 for graphics and input handling.

Features

  - Full CHIP-8 instruction set implementation

  - Graphical display (64x32 pixels) using SDL2

  - Keyboard input (original CHIP-8 hex keypad layout)

  - Adjustable clock speed (CPU frequency)
